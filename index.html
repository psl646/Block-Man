BLOCKMAN

Block Man is trapped in a cave! Help him reach the exit.
Unfortunately, Block Man's legs aren't strong enough to allow him to jump.
But this won't stop Block Man!
Use the blocks and create a path for Block Man to climb on to reach the exit!

    CONTROLS:
  • Left/Right Arrow: Move left/right.
  • Up Arrow: Climb onto a block (when standing next to one).
  • Down Arrow: Pick up block (when standing next to one)
    or drop a block (if holding one).
  • R: Restart Level.
